# Black Store App

Full-stack store recommendation website with:

- React frontend
- Express backend
- SQLite database
- JWT login
- Demand analysis and store location recommendation

## Demo login

- Email: `admin@blackstore.demo`
- Password: `demo123`

## Run locally

1. Install dependencies:
   - `.\setup.ps1`
2. Start dev:
   - `.\run-dev.ps1`

## Main folders

- `client`: React app
- `server`: Express API and SQLite setup
